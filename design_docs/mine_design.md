> 최종 수정일: 2026-03-08

## 1. 개요

### 핵심 컨셉
기지 탭 내 서브 컨텐츠로, 6x6 보드판의 블록을 곡괭이/폭탄/다이너마이트로 파괴하며 광물과 보상을 채굴하고 층을 올라가는 광산 컨텐츠.

### 컨텐츠 목적
- 방치형 게임의 능동적 플레이 요소 제공
- 광물(전용 재화) 획득 경로
- 경험치 파편, 보석, 펫 등 다양한 보상 지급 채널
- 곡괭이 충전 시스템으로 주기적 재방문 유도

### 해금 조건
- `[미정]` 캠페인 스테이지 클리어 조건으로 해금 예정. 구체적 스테이지는 추후 결정

## 2. 기지 탭 진입

### 카드 배치
- 기지 탭("본사") 내 기존 컨텐츠(점성, 세계, 폐허, 연금술, 주점 등)와 동일한 카드 형식
- 카드 구성: 컨텐츠명("광산"), 배경 일러스트, 전용 재화(광물) 아이콘 + 보유 개수(CurrencySlotView), "이동" 버튼

### 기지 탭 확장 방향
- 기지 탭에는 향후에도 동일 형식의 서브 컨텐츠를 계속 추가할 예정

## 3. 화면 구성

### 3.1 상단 HUD
- 골드, 젬(보석), 광물 3종 재화 표시
- 현재 층 표시 (예: "층 7")
- 정보 팝업(i) 버튼 - 터치 시 규칙 설명 팝업 오픈

### 3.2 곡괭이 영역
- 보유량 표시: "50/50" 형식 (현재/최대)
- 최대치일 때 "최대" 텍스트 표시
- 충전 중일 때 다음 충전까지 남은 시간 표시

### 3.3 진행도 보상 슬라이더
- 보드판 상단에 위치
- 층을 올라갈 때마다 슬라이더가 차오름
- 보상 획득 시 슬라이더 UI 전체가 왼쪽으로 이동하며 오른쪽에서 새로운 보상이 등장
- 좌측: 이전 획득 완료 보상(체크 표시), 우측: 다음 목표 보상
- 보상 지급 층은 데이터시트로 관리 (예: 1, 3, 5, 10, 50, 100, ...)
- 터치 시 진행도 보상 팝업 오픈 (전체 보상 목록 스크롤 리스트)

### 3.4 보드판 (6x6)
- 36개 블록이 모두 가려진(바위) 상태로 시작
- 체스판 패턴 배경 (파괴된 칸은 어두운 색으로 구분)
- 블록 터치/사용 시 파괴 애니메이션 후 내부 아이템 표시 (또는 빈칸)
- 상자 발견 시 해당 위치에 보물상자 표시 -> 터치 시 상자 강화 진행
- 상자 획득 완료 후 해당 자리에 다음 층 이동 아이콘(계단) 표시

### 3.5 하단 도구 버튼 (우측)
3개 버튼 가로 배치, 각 버튼에 아이콘 + 보유 개수 표시:

| 도구 | 파괴 범위 | 소모 | 비고 |
|------|----------|------|------|
| 곡괭이 | 1칸 | 곡괭이 1개 | 기본 도구 |
| 폭탄 | 3x3 (최대 9칸) | 폭탄 1개 | 터치 지점 중심 |
| 다이너마이트 | + 모양 (해당 행+열 전체) | 다이너마이트 1개 | 터치 지점 기준 |

- 선택된 도구에 오버레이(선택 상태 표시) 적용
- 곡괭이가 0개면 블록 파괴 불가, 폭탄/다이너마이트는 자체 수량만 차감

### 3.6 하단 기능 버튼 (좌측)
- 나가기 버튼 - 광산에서 기지 탭으로 복귀
- 빠른 클리어 버튼

## 4. 핵심 메커니즘

### 4.1 블록 파괴

#### 기본 플로우
1. 하단에서 도구 선택 (기본: 곡괭이)
2. 보드판 블록을 **꾹 누르기** -> 파괴 예정 영역 시각적 프리뷰 표시
3. **드래그 이동** 가능 -> 프리뷰가 드래그 위치를 따라감
4. 블록 위에서 손을 떼면 -> 해당 영역 파괴 실행
5. 보드판 밖으로 드래그 후 손을 떼면 -> 파괴 취소 (도구 소모 없음)

#### 파괴 결과
- 블록 파괴 애니메이션 재생
- 가중치 테이블에 따라 아이템 출현 또는 빈칸
- 아이템이 있으면 해당 블록 위치에 아이템 아이콘 표시
- 아이템 터치 시 획득 (즉시 지급)
- 상자는 별도 강화 플로우 진행 (4.3 참조)

### 4.2 블록 보상 가중치

#### 보상 종류

| 종류 | 설명 |
|------|------|
| 빈칸 | 아무것도 없음 |
| 광물 (개수별) | 전용 재화, 개수는 층 구간에 따라 변동 |
| 곡괭이 | 곡괭이 수량 회복 |
| 폭탄 | 폭탄 수량 획득 |
| 다이너마이트 | 다이너마이트 수량 획득 |
| 경험치 파편 | 추후 영웅 성장 재료 |
| 보물상자 | 층당 정확히 1개, 상자 강화 플로우 진입 |

#### 가중치 관리
- 데이터시트로 층 구간별 가중치 관리
- 구간 방식: 1층 설정, 10층 설정, 20층 설정 -> 1~9층은 1층 설정 적용, 10~19층은 10층 설정 적용
- 각 구간에서 아이템별 출현 가중치, 광물 개수 범위 등 정의
- 보물상자는 층당 반드시 1개 배치 (36칸 중 1칸에 고정 배치, 나머지 35칸이 가중치 대상)

### 4.3 보물상자 강화

#### 등급 체계
기존 ItemGrade enum 활용 (Advanced ~ Myth):

| 등급 | ItemGrade | 강화 순서 |
|------|-----------|----------|
| 고급 | Advanced (1) | 시작 등급 |
| 희귀 | Rare (2) | 1회 성공 |
| 영웅 | Epic (3) | 2회 성공 |
| 전설 | Legend (4) | 3회 성공 |
| 신화 | Myth (5) | 4회 성공 |

#### 강화 플로우
1. 상자 블록 터치 시 강화 UI 진입
2. 4개의 강화 슬롯 표시 (각 슬롯 상태: 아직 터치 안 함 / 성공 / 실패)
3. 슬롯을 순서대로 터치하면 각각 25% 확률로 성공/실패 판정
4. 성공 시 상자 등급 즉시 상승 + 등급별 이펙트 변경 피드백
5. 실패 시 등급 유지
6. **4번 모두 완료해야** 최종 등급 확정 및 보상 지급
7. 보상은 상자 등급별 데이터시트에서 결정 (등급별 보상 종류/개수 차등)

#### 상자 보상 종류
- 광물, 경험치 파편, 보석, 펫 등
- 등급별 보상 테이블은 별도 데이터시트로 관리

#### 강화 확률
- 매 회차 동일한 25%
- 층에 따른 확률 변동 없음

### 4.4 층 이동

1. 보물상자 강화 완료 및 보상 수령
2. 상자가 있던 위치에 다음 층 이동 아이콘(계단) 표시
3. 계단 터치 시:
   - 파괴했지만 아직 터치하지 않아 남아있던 아이템 **일괄 지급**
   - 다음 층으로 이동
   - 보드판 초기화 (36개 블록 모두 가려진 상태)
   - 새로운 가중치 테이블 적용 (해당 층 구간)

### 4.5 최대 층 도달 (4000층)
- 최대 층에 도달해도 플레이 가능
- 보물상자 획득 시 층 번호는 유지, 보드판만 새로 리셋
- 진행도 보상은 4000층이 마지막

### 4.6 곡괭이 충전 시스템

| 항목 | 값 |
|------|------|
| 최대 보유량 | 50개 |
| 충전 주기 | 15분당 1개 |
| 충전 방식 | 자동 (오프라인에서도 충전) |
| 버프 확장 | 최대 개수 증가, 충전 시간 감소 - 외부 컨텐츠에서 버프 부여, 가능한 구조만 확보 `[추후]` |

### 4.7 빠른 클리어

#### 동작 방식
1. 빠른 클리어 버튼 터치
2. 내부적으로 자동 진행:
   - 랜덤 블록 파괴 (곡괭이 1개 소모/회)
   - 상자 발견 시 자동으로 강화 확률(25%) 4회 계산
   - 상자 보상 자동 획득 + 다음 층 진행
   - 미수령 아이템 자동 획득
3. 보유 곡괭이 전량 소모 시 종료
4. 결과 화면 표시:
   - 소모한 곡괭이 수
   - 클리어한 층 수
   - 획득 보상 목록 (종류별 합산)

## 5. 재화 및 아이템

### 5.1 광물 (전용 재화)
- 광산 컨텐츠 전용 재화
- 획득 경로: 블록 파괴, 보물상자, 진행도 보상
- 사용처: 연구 시스템 `[추후]`
- CurrencySlotView로 표시 (기지 탭 카드, 광산 상단 HUD)

### 5.2 곡괭이
- 커런시로 관리
- 획득 경로: 자동 충전(15분/1개), 블록 파괴 시 획득, 진행도 보상
- 사용처: 블록 파괴 시 소모
- `[추후]` 상점 구매 경로 추가 가능

### 5.3 폭탄
- 커런시로 관리
- 획득 경로: 블록 파괴 시 획득, 진행도 보상
- 사용처: 3x3 범위 블록 파괴
- `[추후]` 상점 구매 경로 추가 가능

### 5.4 다이너마이트
- 커런시로 관리
- 획득 경로: 블록 파괴 시 획득, 진행도 보상
- 사용처: + 모양 범위 블록 파괴
- `[추후]` 상점 구매 경로 추가 가능

### 5.5 경험치 파편
- `[추후]` 영웅 성장 재료로 사용 예정

## 6. 데이터시트 구조

### 6.1 블록 보상 가중치 테이블
- 키: 층 구간 시작 번호 (1, 10, 20, ...)
- 값: 각 아이템 종류별 가중치, 광물 개수 범위

### 6.2 보물상자 보상 테이블
- 키: 상자 등급 (고급/희귀/영웅/전설/신화)
- 값: 보상 종류별 지급 개수, 보상 풀 (광물, 경험치 파편, 보석, 펫 등)

### 6.3 진행도 보상 테이블
- 키: 층 번호 (1, 3, 5, 10, 50, 100, ..., 4000)
- 값: 보상 종류 + 개수

### 6.4 곡괭이 설정 테이블
- 최대 보유량, 충전 주기(초)
- `[추후]` 버프에 의한 변동값

## 7. 정보 팝업

레퍼런스 기준 5개 항목 카드형 설명:

1. 바위를 파괴하면 광석, 경험조각, 보석 등 귀중한 보상을 얻을 수 있습니다
2. 각 방마다 숨겨진 상자 1개를 찾을 수 있습니다. 숨겨진 상자를 열어 다음 방으로 가는 계단을 해제하세요
3. 더 깊이 들어갈수록 더 나은 보상과 방을 발견할 수 있습니다
4. 아이템을 사용하여 바위를 파괴할 수 있습니다: 곡괭이 1개 = 바위 1개, 폭탄 1개 = 3x3 바위, 다이너마이트 1개 = 가로세로 바위
5. 곡괭이는 자동으로 충전됩니다
